# boost-tools

Tools copied from Boost official release.

bash (for Linux) or git bash (for Windows):

```bash
make prepare
cd build
cmake ..                    # Windows: cmake -G "Visual Studio 14 2015 Win64" ..
make                        # Windows: open the *.sln file with VS2015
```
